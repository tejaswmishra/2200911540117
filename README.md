# Backend Project

This is a **Node.js backend project** demonstrating APIs and middleware.

---

## **Project Setup**

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd backend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the server:
   ```bash
   node index.js
   ```
   Or, if using Nodemon:
   ```bash
   npm start
   ```
